# Methodology

## Approach

A hybrid security assessment approach was used:

1. Static Analysis (SAST)
2. Dynamic Analysis (DAST)
3. Threat Modelling

## Tools Selection

### Horusec

* Multi-language support
* CLI + JSON output
* Suitable for automation 

### OWASP ZAP

* Active + passive scanning
* Real-time vulnerability detection
* Severity classification 

## Workflow

1. Setup vulnerable app
2. Run SAST scan
3. Run DAST scan
4. Aggregate results
5. Perform threat modelling
6. Rank risks
